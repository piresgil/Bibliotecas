Projects!

Keep in mind, these are just projects. They are not finished products, but mostly just stuff I am working on that looked cool enough to upload at the time. The code is not optimised at all, but the concepts are cool.


Have Some JavaScript Projects!

These links open the projects directly in your browser.


Noise Topography - Generates 1D value noise as well as a 2D value noise infinite terrain. Performance can be greatly improved.


Simplex Noise - Generates 1D and 2D simplex noise, which is much smoother than value noise. Again, I threw this together quickly and it can be improved.


Isometric Tile - Super cool pseudo 3D tile plotter that will crash your PC if it isn't lightning fast.


Orthographic 3D - An orthographic 3D engine written completely from scratch in JavaScript, running without WebGL or any other libraries. It does not use a Z buffer, so there is a little bit of z fighting between surfaces. This is not meant for use in production.


Rasterizer - A Pure JS rasterizer. It plots points in 2D shapes. Essentially it is a graphics drawing API that gives you access to each and every pixel in a shape before it is drawn. This is the basis for z buffering.


Have Some PowerShell Projects!

These links will download Powershell Scripts to your computer.


Stay Awake - A PS script that enters a keystroke every several seconds to keep your computer from entering sleep mode.


Idle Alert - A PS script that will cause a terminal window to flash red and white if there is no mouse activity for over 10 minutes, reminding you to move the mouse before your screen falls asleep.
